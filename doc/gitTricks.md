<h1>Git Command Line Tricks</h1>

## Branch

* check current branch

```dos
git branch
git checkout <branch name>
```

## Status

```dos
git log --oneline
```

## Pull without merge

```dos
git reset
git pull
```

## config user

```dos
git config user.name "jwang1122"
git config user.email "jwang1122.gmail.com"
```
